## 解題說明

 problem1要求計算Ackermann函數，需要遞迴和非遞迴兩種做法。

### 解題策略
Ackermann 函數 \( A(m, n) \) 的數學定義如下：

$$
A(m, n) = 
\begin{cases} 
n + 1 & \text{if } m = 0 \\
A(m - 1, 1) & \text{if } n = 0 \\
A(m - 1, A(m, n - 1)) & \text{otherwise}
\end{cases}
$$

在遞迴寫法下:
1. 輸入數值進入遞迴函式。
2. 會依照if條件判斷目前的狀況，並做出相對應的計算，當m=0時，回傳n+1。
3. 當最內層的遞迴結束，程式會依照順序繼續回傳結果。
4. 主程式收到函式的最後的回傳值，並將結果輸出。
